###프로젝트명: COMMENTOON (웹툰 + 댓글)
######[prestudy-toyproject]

<p> ✅ "COMMENTOON" 은 국내 최대 웹툰 사이트인 ‘NAVER WEBTOON’을 크롤링하여 사용자들의 의견을 교류할 수 있는 사이트입니다. </p>
<p> ✅ 웹툰을 읽는 도중,
혹은 읽은 직후에 자신이 느낀 감정을 다른 이들과 함께 댓글로 교류할 수 있습니다!</p>
